Type & Size

Type & Size permite a sus usuarios realizar algunas operaciones que juegan con el estilo y el tamaño de los textos que estos introduzcan. La respues de Type & Size se pueden ver al momento.

¿Cómo usar Type & Size?

No debes preocuparte por cómo usar esta aplicación ya que ofrece un diseño muy intuitivo y sencillo de usar, solo debes introducir un texto y escoger una de las opciones que ofrece la app.
